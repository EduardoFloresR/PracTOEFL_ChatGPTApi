What is Prac-TOEFL?

Prac-TOEFL is a project developed by Eduardo Flores for the subject of Selected Programming Topics by the UNAM Faculty of Engineering.

The web application was made using the Python programming language and the Flask library. It also uses the SQLAlchemy library to manage and register users in a database.

The analysis of the implemented texts is carried out through the ChatGPT API, which was configured as an English language instructor for TOEFL preparation.
